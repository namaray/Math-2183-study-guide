1. Differential Equations (Constant Coefficients)

To solve equations of the form $$ay'' + by' + cy = g(x)$$:

Homogeneous Part ($y_c$)

Solve the auxiliary equation: $$am^2 + bm + c = 0$$[cite: 251, 257].

  • Distinct Real Roots: $y = C_1e^{m_1x} + C_2e^{m_2x}$[cite: 265].
  • Repeated Real Roots: $y = C_1e^{m_1x} + C_2xe^{m_1x}$[cite: 266].
  • Complex Roots: $y = e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)$[cite: 287].

Non-Homogeneous Part ($y_p$)

The general solution is $y = y_c + y_p$[cite: 152]. Use the method of undetermined coefficients for $y_p$ based on $g(x)$[cite: 155, 157]:

$g(x)$ Form Trial $y_p$ Guess
Constant (1)$A$
$5x+7$$Ax+B$
$\sin 4x$ / $\cos 4x$$A \cos 4x + B \sin 4x$
$e^{5x}$$Ae^{5x}$

2. Linear Systems and Matrix Rank

Analyze $Ax = b$ by comparing $rank(A)$ and $rank([A|b])$ [cite: 521, 907-911].

• Inconsistent: $rank(A) \neq rank([A|b])$[cite: 524, 912].
• Unique: $rank(A) = rank([A|b]) = n$[cite: 531, 917].
• Infinite: $rank(A) = rank([A|b]) < n$[cite: 535, 915].

4. Eigenvalues & Eigenvectors

Find $\lambda$ by solving $\det(A - \lambda I) = 0$[cite: 129, 134].

  • $tr(A) = \sum \lambda_i$[cite: 135].
  • $\det(A) = \prod \lambda_i$[cite: 135].
