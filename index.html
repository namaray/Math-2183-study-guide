1. Differential Equations

To solve equations of the form:

$$ay'' + by' + cy = g(x)$$ [cite: 248]

Homogeneous Part (\(y_c\))

Solve the auxiliary equation: $$am^2 + bm + c = 0$$ [cite: 251, 257]

  • Distinct Real Roots: \(y = C_1e^{m_1x} + C_2e^{m_2x}\) [cite: 265]
  • Repeated Real Roots: \(y = C_1e^{m_1x} + C_2xe^{m_1x}\) [cite: 266]
  • Complex Roots (\(\alpha \pm i\beta\)): \(y = e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)\) [cite: 287]

Non-Homogeneous Part (\(y_p\))

General solution: \(y = y_c + y_p\) [cite: 152]. Use Undetermined Coefficients based on \(g(x)\) [cite: 155, 157]:

Shape of \(g(x)\) Guess for \(y_p\)
Constant (e.g., 6)\(A\)
Polynomial (e.g., \(30x+3\))\(Ax + B\)
Trig (\(\cos 2x\))\(A \cos 2x + B \sin 2x\)
Exponential (\(e^{-2x}\))\(Ae^{-2x}\)

2. Linear Systems & Rank

Consistency rules for \(Ax = b\) with \(n\) unknowns [cite: 521, 907-911]:

  • Inconsistent (No Solution): \(rank(A) \neq rank([A|b])\) [cite: 527, 912]
  • Consistent (Unique): \(rank(A) = rank([A|b]) = n\) [cite: 531, 917]
  • Consistent (Infinite): \(rank(A) = rank([A|b]) < n\) [cite: 535, 539, 914-915]

4. Eigenvalues & Trace

Solve \(\det(A - \lambda I) = 0\) to find \(\lambda\) [cite: 129, 134].

Trace Relationship: \(tr(A) = \lambda_1 + \lambda_2\)
Determinant Relationship: \(\det(A) = \lambda_1 \cdot \lambda_2\) [cite: 135]
