Linear Differential Equations Study Guide

A comprehensive guide to Homogeneous and Non-Homogeneous equations with constant coefficients.

1. Overview & Classification

To determine which method to use, look at the right-hand side of the standard equation:
$$a_n y^{(n)} + \dots + a_1 y' + a_0 y = g(x)$$

[cite_start] [cite_start]
Type Condition Goal
Homogeneous $g(x) = 0$Find the Complementary Function ($y_c$) using roots of the auxiliary equation[cite: 162].
Non-Homogeneous $g(x) \neq 0$Find the General Solution ($y$) by adding $y_c$ and a Particular Solution ($y_p$): $$y = y_c + y_p$$[cite: 16].

2. Topic I: Homogeneous Equations

[cite_start]

Core Concept: Solve $ay'' + by' + cy = 0$ by finding the roots ($m$) of the auxiliary equation $am^2 + bm + c = 0$ [cite: 111-115].

Case I: Distinct Real Roots ($m_1 \neq m_2$)

Formula: $$y_c = c_1 e^{m_1 x} + c_2 e^{m_2 x}$$

Case II: Repeated Real Roots ($m_1 = m_2 = m$)

Formula: $$y_c = c_1 e^{mx} + c_2 x e^{mx}$$
[cite_start]

Note: If the root repeats 3 times, add a $c_3 x^2 e^{mx}$ term, and so on[cite: 242].

Case III: Conjugate Complex Roots ($m = \alpha \pm i\beta$)

Formula: $$y_c = e^{\alpha x}(c_1 \cos \beta x + c_2 \sin \beta x)$$
[cite_start]

Identify the real part ($\alpha$) and the imaginary part ($\beta$) from the root[cite: 132, 151].

3. Topic II: Non-Homogeneous Equations

[cite_start]

Core Concept: To solve $ay'' + by' + cy = g(x)$, first find $y_c$, then find a particular solution $y_p$ using the Method of Undetermined Coefficients[cite: 19].

Step 1: Select Your Trial Solution ($y_p$)

[cite_start]
If $g(x)$ contains...Try this Trial Solution ($y_p$) [cite: 22]
Constant (e.g., 5) $A$
Linear (e.g., $5x + 7$) $Ax + B$
Quadratic (e.g., $3x^2 - 2$) $Ax^2 + Bx + C$
Exponential (e.g., $e^{5x}$) $Ae^{5x}$
Sin / Cos (e.g., $\sin 4x$) $A \cos 4x + B \sin 4x$

Step 2: The "Glitch" Rule (Duplicates)

[cite_start]

If any term in your chosen $y_p$ is already present in your complementary solution $y_c$, you must multiply that part of $y_p$ by $x$ (or $x^2$ if $x$ is also taken)[cite: 64].

Example: If $g(x) = e^x$ but $y_c$ has $c_1 e^x$, use $y_p = Axe^x$.

Step 3: Solve for Coefficients

    [cite_start]
  1. Differentiate $y_p$ to find $y_p'$ and $y_p''$ [cite: 36-38].
  2. Substitute them back into the original DE.
    3. [cite_start]
  3. Equate coefficients of like terms to solve for constants $A, B, C$[cite: 41].

4. Summary Checklist

  • ✅ Check RHS: Is it $0$ (Homogeneous) or $g(x)$ (Non-Homogeneous)?
  • ✅ Find $y_c$: Solve $am^2 + bm + c = 0$.
  • ✅ Find $y_p$ (if needed): Guess form based on $g(x)$.
  • ⚠️ Watch Out: Multiply $y_p$ by $x$ if it overlaps with $y_c$.
    • [cite_start]
  • ✅ Final Answer: Combine $y = y_c + y_p$[cite: 16].
