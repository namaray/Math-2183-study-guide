Rank of Matrix & Systems of Equations

A study guide for determining system consistency using Rank arguments, based on the uploaded notes.

1. Core Definitions

To analyze a system $Ax=b$, you must compare the ranks of two specific matrices.

Key Terms:

  • Rank ($r$): The number of non-zero rows in the row echelon form of a matrix.
  • $n$: The number of unknowns (variables) in the system.
  • Coefficient Matrix ($A$): The matrix containing only the coefficients of variables.
  • Augmented Matrix ($[A|b]$): The matrix $A$ with the constants vector $b$ attached as the last column.

2. Classification of Solutions

Use Gaussian Elimination to find the rank, then apply these three cases to determine the nature of the solution.

Case Rank Condition Result
Inconsistent $\text{rank}(A) \neq \text{rank}([A|b])$ No Solution.
(Usually occurs when a row reduces to $0=c$ where $c \neq 0$)
Consistent (Unique) $\text{rank}(A) = \text{rank}([A|b]) = n$ Unique Solution.
(Number of independent equations equals number of variables)
Consistent (Infinite) $\text{rank}(A) = \text{rank}([A|b]) < n$ Infinitely Many Solutions.
(There are $n - r$ free variables).

3. Worked Examples & Analysis

Example A: Finding Consistency (Problem 7)

System: Last row reduces to $[0 \quad 0 \quad 0 \mid k-3]$.
Goal: Find $k$ for consistency.

Analysis:

  • Rank of $A$ is 1 (from previous rows).
  • For consistency, we need $\text{rank}([A|b])$ to also be 1.
  • This requires the last term to be zero: $k-3 = 0 \implies k=3$.

Conclusion:

  • If $k=3$: Consistent. Since $r=1$ and $n=3$, there are infinite solutions ($3-1=2$ free variables).
  • If $k \neq 3$: Inconsistent (No solution).

Example B: Parameter Analysis (Problem 8)

System: Last row reduces to $[0 \quad 0 \quad 4-a \mid 1]$.
Goal: Classify based on $a$.

1. No Solution ($a=4$):
If $a=4$, the row becomes $[0 \quad 0 \quad 0 \mid 1]$.
This means $0 = 1$, which is impossible.
$\text{rank}(A)=2 \neq \text{rank}([A|b])=3$.

2. Unique Solution ($a \neq 4$):
If $a \neq 4$, the term $(4-a)$ is non-zero.
Rank is 3, and $n=3$. Since $r=n$, the solution is unique.

3. Infinite Solutions (Never):
To have infinite solutions, we need a row of all zeros $(0=0)$.
However, the augmented part is always $1$. We can never get $0=0$ in the last row, so this case is impossible.

4. Summary Checklist

  • ✅ Form Augmented Matrix: Write $[A|b]$.
  • ✅ Row Reduce: Get it into Row Echelon Form.
  • ✅ Count Non-Zero Rows: Find $\text{rank}(A)$ (left side) and $\text{rank}([A|b])$ (whole matrix).
  • ✅ Compare:
    • Different ranks? $\to$ No Solution.
    • Same rank ($r$) equals $n$? $\to$ Unique.
    • Same rank ($r$) less than $n$? $\to$ Infinite ($n-r$ free variables).
